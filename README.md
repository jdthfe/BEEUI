# 一套基于vue2.0+ui组件库

# ui组件源码主要基础框架依赖
vue 2.5.8

# 文档主要基础框架依赖
vue 2.5.8
vue-router 2.8.1

# 合并打包命令（文档打包/lib打包/主题样式打包）

``` bash
# 打包js/css命令
npm run lib
```

# 运行命令

``` bash
# 本地测试
npm run dev

# 本地打包dsit文档静态文件
npm run build

# 本地打包lib文件夹下的样式
npm run lib:style

# 本地打包lib文件夹下的主题
npm run lib:theme

# 本地打包lib文件夹下的js
npm run lib:js

# 组件测试打包命令
npm run unit

```
